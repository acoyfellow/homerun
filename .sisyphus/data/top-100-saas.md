## Usage

This list is used by the unsurf automated scouting pipeline to populate the API directory. Each tool is categorized by:

- **Category**: Functional area
- **Difficulty**: Estimated complexity for scouting (easy/medium/hard)
- **Description**: What the tool does

## Contributing

To add a new tool:
1. Add entry to `top-100-saas.json`
2. Regenerate this markdown file
3. Submit PR with description of why this tool is important
